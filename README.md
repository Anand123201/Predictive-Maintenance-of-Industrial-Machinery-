📘 Predictive Maintenance of Industrial Machinery
📌 Overview

This project focuses on building a predictive maintenance system using machine learning to detect potential failures in industrial machinery. The model is developed using IBM Watsonx AutoAI, which automates model selection, training, and optimization.

The aim is to reduce unexpected machine breakdowns and improve operational efficiency.

🎯 Objectives
Predict equipment failures using historical data
Automate model building using AutoAI
Compare multiple ML pipelines
Deploy the best-performing model for real-time predictions
🛠️ Tech Stack
Programming Language: Python
Libraries/Tools: Scikit-learn, XGBoost, LightGBM
Platform: IBM Watsonx AutoAI
Environment: Jupyter Notebook
